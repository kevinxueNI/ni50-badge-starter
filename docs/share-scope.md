# Share Scope

## Purpose

This repository is the minimum clean share package for the NI50 badge project.

It is designed for private read-only sharing.

## Included

- `boards/2.8-inch/project/`
- `boards/4-inch/project/`
- board-level README files
- quick-start guidance
- starter Copilot prompts
- read-only sharing notes

## Excluded

The following are intentionally not included:

- wiki pages
- raw source folders
- private vault content
- local build outputs
- downloaded `managed_components`
- local logs and temporary files
- unrelated historical working notes

## Why The Scope Is Narrow

The goal is not to expose the entire working repository.

The goal is to provide:

1. a clean code entry point,
2. enough context to build and modify the project,
3. a low-noise starter package that is easy to browse.
